# SwitchBot
[SwitchBot](https://www.switch-bot.com/) api for Node.js (unofficial)

# Usabe

```
const Switchbot = require('switchbot');
const switchbot = Switchbot('bluetooth address');
switchbot.turnOn();
switchbot.turnOff();
switchbot.press();
```

# Depends
gatttool is required
